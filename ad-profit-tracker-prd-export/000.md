ABOUT THIS PROJECT
------------------

Build a full-stack web app called “Ad Profit Tracker” using Node.js backend,
React frontend, and Firebase for authentication, database, and secure secret
storage. The app should have: - Firebase login with Google or email/password -
A protected Settings page first, where each user can save their own API keys
securely - Modular network setup so ad networks can be added or changed easily
later The app must track: - ExoClick = cost only - RollerAds = revenue only -
Zeydoo = revenue only - Propush = revenue only Show: - Total Revenue - Total
Cost - Net Profit - ROI % - Daily profit trend chart - Country/GEO breakdowns -
Per-network tabs with tables and charts - API Explorer tab showing what each
API returns - Export to Excel and PDF - Refresh All Data Now button - Automatic
daily data pull Use these API sources: - ExoClick stats with country grouping -
RollerAds `/sites/stat` using ACCESS_TOKEN - Zeydoo CPA statistics endpoint -
Propush `/v5/pub-statistics` with geo grouping Security requirements: - Never
expose API keys in frontend code - Store historical data in Firestore - Keep
everything deployable to Cloud Run Start by building the login system, then the
Settings page, then the dashboard and data fetching.

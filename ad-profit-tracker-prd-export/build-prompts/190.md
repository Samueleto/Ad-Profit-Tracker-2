Before building the two main components (GeoBreakdownSection and GeoCountryDrilldownModal), we need a handful of smaller pieces that both of them share or that would be too complex to keep inline. Build these first so the main components can import them by name.

**MetricShareBar** — A thin inline progress bar component. Takes a value (0–100) and renders a filled bar inside a light gray track, all within a fixed-width container that fits in a table cell. Keep it compact — this sits in a dense table row. No labels, just the visual bar.

**GeoColorRow** — A wrapper that applies the project-wide Tailwind semantic color classes to a table row based on the colorCode field from GET /api/roi/breakdown. Maps 'positive' → bg-green-50/text-green-600, 'negative' → bg-red-50/text-red-600, 'neutral' → bg-amber-50/text-amber-600. Takes colorCode as a prop and applies the right classes to its children.

**GeoInsightsStrip** — The collapsible strip that sits below the country table in GeoBreakdownSection. Takes four props: topCountry (GeoCountryRow), worstCountry (GeoCountryRow), positiveRoiCount (number), totalCountries (number). Renders collapsed by default with a chevron toggle. When expanded, shows the four stats in a compact horizontal layout. The chevron rotates on expand/collapse.

**NetworkSubRow** — The inline expandable sub-row that appears below a country row when the user clicks the 'Network Breakdown' toggle. Takes an array of GeoNetworkContribution objects and renders them as a compact indented table with columns: network name, data role badge, primary metric value (formatted as currency or '—' if null), and percentage of total. This sub-row spans the full width of the parent table.

**GeoTableSkeleton** — A loading placeholder that renders 10 shimmer rows matching the exact column structure of the country table (flag+name, revenue, cost, profit, ROI%, impressions, clicks, share bar). Used by GeoBreakdownSection during its loading state. Each shimmer cell should match the approximate width of its real counterpart.

**CountryTrendChart** — A recharts LineChart (200px height, full width of its container) that takes an array of GeoSnapshotDayPoint objects and plots net profit per day. X-axis shows dates, Y-axis shows profit values. Handles null netProfit values gracefully (skip the point rather than plotting zero). Used inside GeoCountryDrilldownModal. Mark this component 'use client' since it uses recharts.

All components should use the TypeScript types defined in the geo-breakdown types file. Keep them purely presentational where possible — no data fetching inside these sub-components.

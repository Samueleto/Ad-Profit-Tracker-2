We need several reusable components that both PerNetworkAnalyticsTabsSection and ApiExplorerTab will use. Build these before the pages so the pages can import them by name.

**NetworkTabBar** — The tab bar that sits at the top of the section. Takes an array of network configs and renders a tab for each: ExoClick, RollerAds, Zeydoo, Propush, and API Explorer. Each network tab shows the network name plus a status dot — green if `lastSyncStatus` is 'success', red if 'failed', gray if null/never synced. The active tab is highlighted. Clicking a tab fires an `onTabChange` callback with the tab id. This component is purely presentational — no data fetching.

**NetworkKpiRow** — A row of 4 stat cards shown at the top of each network tab panel. Takes a `summary` object (from `NetworkStatsResponse`) and a `dataRole` ('cost' or 'revenue'). Renders cards for: primary metric (labeled 'Cost' for ExoClick, 'Revenue' for others), impressions, clicks, and CTR%. Each card shows the value and a label. In loading state, renders 4 shimmer placeholder cards. Null/undefined values render as '—'.

**NetworkTrendChart** — A recharts `AreaChart` wrapped in `ResponsiveContainer` at 280px height. Takes a `series` array from the daily groupBy response. The area fill is amber (`#F59E0B`) for cost networks (ExoClick) and green (`#10B981`) for revenue networks. Includes `XAxis` (date), `YAxis`, `CartesianGrid`, `Tooltip` with formatted values, and a horizontal `ReferenceLine` at y=0 as the breakeven line. Shows a 280px shimmer rectangle in loading state. Shows a centered 'No trend data available' message if series is empty.

**NetworkGeoTable** — A compact table showing country-level breakdown. Takes a `countries` array from the country groupBy response. Columns: flag emoji + country name, primary metric, impressions, clicks, CTR%, and a MetricShareBar (reuse from GeoBreakdownSection). Rows are color-coded using the `colorCode` field — green for positive, red for negative, amber for neutral. Clicking a row fires an `onCountryClick` callback with the country code (to open the existing GeoCountryDrilldownModal). Shows 8 shimmer rows in loading state. Accepts a `topN` prop to limit visible rows (wire up to TopNSelector from GeoBreakdownSection).

**NetworkInsightsStrip** — A collapsible strip below the GEO table. Collapsed by default with a chevron toggle. When expanded, shows: best country (highest primary metric), worst country (lowest), total countries tracked, and last sync timestamp. Also contains the 'Sync This Network' button that fires an `onSync` callback with the networkId. The button shows a loading spinner while a sync is in progress.

**ApiExplorerAccordion** — A single accordion section for one network in the API Explorer tab. Takes a `network` prop. Shows the API endpoint URL and HTTP method, required parameters list, a syntax-highlighted JSON code block for the raw response (use `<pre>` with monospace styling and a copy-to-clipboard button — no need for a heavy syntax highlighting library, just JSON.stringify with indentation), a field mapping table (two columns: API field → App field with type annotations), a 'Fetch Fresh Sample' button that fires an `onFetchSample` callback, and a 'Last fetched' timestamp. If no API key is configured, shows 'API key not configured — go to Settings' with a link to /settings instead of the JSON viewer. Shows a shimmer in the JSON area while loading.

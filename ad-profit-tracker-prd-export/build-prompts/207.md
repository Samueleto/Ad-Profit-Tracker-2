We need several reusable components that the ApiExplorerTab will assemble together. Build these in `src/features/api-explorer/components/`.

**NetworkAccordion** — A collapsible accordion section for a single ad network. Takes a `networkId` (NetworkId), `displayName`, and `children` as props. Collapsed by default. Shows the network name in the header with a chevron icon that rotates on open/close. All four network sections in the ApiExplorerTab will use this component.

**NetworkConfigPanel** — Renders the reference info for a network: the endpoint URL (in a monospace code block), the HTTP method as a badge (GET in blue, POST in green), and a table of required parameters with name and description columns. Takes a `NetworkConfig` object as a prop. If the config hasn't loaded yet, show a shimmer skeleton.

**JsonResponseViewer** — The syntax-highlighted JSON viewer. Uses `react-json-view` (dynamically imported with `ssr: false`). Takes a `data` prop (the raw JSON object) and a `fetchedAt` timestamp string. Shows the JSON tree collapsed to 2 levels by default with expand/collapse controls. Below the viewer, show a 'Last fetched: [relative time]' label. Includes a 'Copy to Clipboard' button that copies the full JSON string and shows a brief 'Copied!' confirmation. When `data` is null and `isLoading` is true, show a shimmer placeholder in place of the viewer. When `data` is null and `isLoading` is false, show the 'No response cached yet. Click Fetch Fresh Sample to retrieve one.' empty state message.

**FetchSampleButton** — A button that triggers the POST to `/api/networks/config/test-connection`. Takes `networkId` and an `onSuccess` callback. Shows a spinner and disables itself while the request is in flight. On error, shows an inline error message below the button with a 'Retry' link. On success, calls `onSuccess` with the fresh response data.

**FieldMappingTable** — A two-column table showing how raw API fields map to the app's internal `adStats` schema fields. Takes an array of `FieldMapping` objects. Columns are 'API Field' and 'Internal Field (adStats)'. If a mapping has notes, show a small info icon with a tooltip. This is purely presentational — no loading state needed since it uses the static `FIELD_MAPPINGS` constant.

**ApiKeyMissingBanner** — A simple informational banner shown when `configured: false` comes back from the endpoint. Shows 'API key not configured' with a link that navigates to `/settings`. Used inside the accordion body in place of the normal content.

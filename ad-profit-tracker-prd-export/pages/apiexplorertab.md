## Description

FUNCTIONALITY: An 'API Explorer' tab integrated into the FinancialMetricsDashboardPage tab bar alongside the four network tabs. Displays a read-only reference panel showing what each ad network API returns. Four collapsible accordion sections — one per network (ExoClick, RollerAds, Zeydoo, Propush). Each section shows: (1) the API endpoint URL and HTTP method used by the app; (2) required parameters (date range, groupBy, access token field name); (3) a live 'Last Raw Response' viewer — a syntax-highlighted JSON code block showing the most recent raw API response fetched from GET /api/networks/{network}/raw-response; (4) response field mapping table showing how raw API fields map to the app's internal adStats schema fields (e.g., 'impressions' → 'impressions', 'revenue' → 'revenue'); (5) a 'Fetch Fresh Sample' button that triggers a lightweight test call via POST /api/networks/config/test-connection for that network, displaying the response inline. A 'Copy Response' button copies the JSON to clipboard. A 'Last fetched' timestamp shows when the raw response was last retrieved. If no API key is configured for a network, shows 'API key not configured — go to Settings' with a link to /settings. The section is purely informational — no data mutations except the test connection call.
DESIGN APPROACH: Accordion layout — all four networks collapsed by default, expand on click. Syntax-highlighted JSON viewer (using a lightweight JSON formatter) with line numbers and copy button. Field mapping table uses a two-column layout (API field → App field) with type annotations. Dense but readable — monospace font for JSON, sans-serif for mapping table. Progressive disclosure — raw response hidden until user expands the accordion.
STATES: loading — accordion headers render immediately, JSON viewer shows 'Loading last response...' shimmer; empty (no API key) — 'API key not configured' message with Settings link inside the accordion; error_401 — attempt token refresh, if fails redirect to /; error_403 — show 'Access Denied' inline; error_404 — 'No response cached yet. Click Fetch Fresh Sample to retrieve one.'; error_500 — show error inside accordion with retry; success — syntax-highlighted JSON rendered, field mapping table populated, last fetched timestamp shown.
SECURITY: Protected route — inherits FinancialMetricsDashboardPage auth. Raw API responses are fetched server-side and never expose decrypted API keys. Bearer token on all calls. Read-only except for test-connection.

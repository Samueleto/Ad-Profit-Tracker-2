## Description

FUNCTIONALITY: A modal dialog triggered by the 'Export' button already present in FinancialMetricsDashboardPage. Users configure and initiate Excel exports. The modal presents two export format tabs — 'Excel (.xlsx)' and 'PDF' — with Excel as the default. For Excel, users choose which sheets to include via checkboxes: Summary (KPI totals), Daily Trend (date-series rows), Geographic Breakdown (country rows), Per-Network Stats (one sheet per network), and Activity Log (audit entries). A date range display shows the currently active dashboard date range (read from Zustand store — no independent selector). A 'Sheet Preview' count badge shows estimated row count per sheet. An 'Export Now' button triggers POST /api/export/excel, which streams back a .xlsx file download. A progress indicator (indeterminate spinner) appears while the server generates the file. On success, the browser triggers a file download and a 'Export complete' toast appears. A 'Cancel' button closes the modal without action. If no data exists for the selected range, the Export Now button is disabled with tooltip 'No data available for this range'.
DESIGN APPROACH: Centered modal (max-w-lg) with backdrop overlay using existing ThemeProvider tokens (dark:bg-gray-900/bg-white). Sheet selection uses compact checkbox rows with row count badges. Progressive disclosure — advanced options (custom filename, include headers toggle) collapsed under 'Advanced' chevron. Export Now button uses accent color CSS custom property. Consistent with project-wide Tailwind semantic classes.
STATES: loading — modal body shows 5 shimmer checkbox rows while row counts load from Zustand store; exporting — Export Now button shows spinner and is disabled, all checkboxes disabled, progress message 'Generating your Excel file...'; empty — Export Now button disabled with tooltip 'No data for selected range'; error_401 — close modal and attempt Firebase ID token refresh via getIdToken(true), if fails redirect to / with 'Session expired' toast; error_403 — show 'Access Denied' inside modal with close button; error_404 — disable Export Now with 'No data found for this date range'; error_500 — show inline error banner inside modal with 'Retry' button; success — file download triggers automatically, toast 'Export complete — file downloaded', modal closes after 1.5s.
SECURITY: Protected — inherits FinancialMetricsDashboardPage Firebase Auth middleware. Bearer token sent with POST /api/export/excel. userId always from verified Firebase ID token server-side. Export scoped to authenticated user's data only.

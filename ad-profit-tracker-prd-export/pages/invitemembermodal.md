## Description

FUNCTIONALITY: A modal dialog triggered by the 'Invite Member' button on TeamManagementPage. Contains an email input field (validates RFC 5322 format on blur), a role selector (Admin or Member — Owner role is not assignable via invite), and an optional personal message textarea (max 200 chars). A 'Send Invitation' button calls POST /api/team/invitations. On success, the modal closes, the pending invitations list refreshes, and a toast 'Invitation sent to [email]' appears. Duplicate email detection (409) shows 'This person is already a member or has a pending invitation' inline below the email field. Rate limit (429) shows 'Too many invitations sent. Try again in an hour.' A 'Cancel' button closes the modal without action. The modal shows the workspace name as context at the top so the invitee knows which workspace they are joining.
DESIGN APPROACH: Centered modal (max-w-md) with backdrop overlay using existing ThemeProvider tokens. Compact vertical form — email at top, role selector below, optional message at bottom. Role selector uses a segmented control (Admin / Member) with brief descriptions of each role's permissions shown as muted helper text. Send button uses accent color CSS custom property. Destructive cancel is ghost-styled.
STATES: loading — Send button shows spinner and is disabled, all inputs disabled; empty (initial open) — form shows defaults (empty email, Member role selected, empty message); error_401 — close modal, call getIdToken(true), if refresh fails redirect to / with 'Session expired' toast; error_403 — show 'You do not have permission to invite members' inside modal with close button; error_404 — show 'Workspace not found' inside modal; error_500 — inline error banner inside modal with Retry; success — modal closes, parent list refreshes, success toast shown.
SECURITY: Protected — inherits SettingsPage Firebase Auth middleware. Bearer token on POST /api/team/invitations. Server verifies inviting user's workspaceRole is Admin or Owner before processing. Rate limited to prevent invite spam.

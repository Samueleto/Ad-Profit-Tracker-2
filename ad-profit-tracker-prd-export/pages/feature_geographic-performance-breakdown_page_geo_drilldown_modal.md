## Description

FUNCTIONALITY: A slide-over modal (right-side drawer, 480px wide on desktop, full-screen on mobile) that opens when a user clicks any country row in the GeoBreakdownSection. Displays the selected country's name and flag at the top. Shows a daily trend mini-chart (recharts LineChart, 200px height) plotting net profit per day for the selected date range for that specific country. Below the chart, a per-network breakdown table shows each network's contribution for that country: network name, data role (Cost Only / Revenue Only), primary metric value, and percentage of total. A 'Date Range' label shows the active range. A 'View Full Trend' link navigates to the DailyProfitTrendSection with the country pre-filtered. Close button (X) in the top-right corner dismisses the modal. Data is fetched via GET /api/stats/snapshot with groupBy='country' filtered to the selected country code for each day in the range — the component iterates over the date range and aggregates. If no data exists for the country in the range, shows 'No data available for [Country] in this period' with a close button.
DESIGN APPROACH: Right-side slide-over drawer with backdrop overlay. Compact but information-dense — mini chart at top, network breakdown table below. Country name and flag as the modal title. Minimal chrome — just close button and content. Smooth slide-in animation. Mobile: full-screen sheet with drag-to-dismiss.
STATES: loading — mini chart area shows 200px shimmer rectangle, network table shows 4 shimmer rows; empty — 'No data available for [Country] in this period' with close button; error_401 — close modal and redirect to / with 'Session expired' toast; error_403 — show 'Access Denied' inside modal; error_404 — show 'No data found for this country' inside modal; error_500 — show error banner inside modal with Retry; success — mini trend chart rendered, per-network breakdown table populated.
SECURITY: Protected — inherits parent page auth. Bearer token on all API calls. No new auth surface.

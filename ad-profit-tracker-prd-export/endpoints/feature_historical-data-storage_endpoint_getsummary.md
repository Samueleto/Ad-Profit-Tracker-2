# Feature Historical Data Storage Endpoint GetSummary

GET /api/stats/summary

## Description

REQUEST: GET with query params: dateFrom (YYYY-MM-DD, required), dateTo (YYYY-MM-DD, required).
PROCESS:
1. Verify Firebase ID token — return 401 if invalid. uid from verified token only.
2. Validate dateFrom and dateTo: required, valid YYYY-MM-DD, dateFrom <= dateTo, max 90-day range. Return 400 if invalid.
3. Query adStats Firestore collection: filter userId==uid AND date>=dateFrom AND date<=dateTo. Use getDocs.
4. Aggregate in memory across all networks and all dates:
   - totalRevenue: sum of revenue fields from rollerads+zeydoo+propush rows
   - totalCost: sum of cost fields from exoclick rows
   - netProfit: totalRevenue - totalCost
   - roi: (netProfit / totalCost * 100) if totalCost > 0 else null
   - totalImpressions: sum across all networks
   - totalClicks: sum across all networks
   - averageCtr: weighted average ctr
   - daysWithData: count of unique dates with at least one record
   - networksWithData: array of networkIds that have at least one record in range
5. Also compute per-network subtotals: for each of the 4 networks, sum their respective primary metric (cost for exoclick, revenue for others) across the date range.
6. Fetch networkConfigs documents for this user (getDocs, filter userId==uid) to include lastSyncedAt and lastSyncStatus per network in the response.
7. Return the aggregated summary.
SUCCESS (200): { dateFrom: string, dateTo: string, totalRevenue: number, totalCost: number, netProfit: number, roi: number|null, totalImpressions: number, totalClicks: number, averageCtr: number, daysWithData: number, networksWithData: string[], perNetwork: [{ networkId: string, dataRole: string, primaryMetric: number, lastSyncedAt: timestamp|null, lastSyncStatus: string }] }
ERRORS:
- 400: missing/invalid dates, range exceeds 90 days
- 401: unauthenticated
- 500: Firestore read failure
SECURITY: userId always from verified token. All queries scoped to authenticated user only.

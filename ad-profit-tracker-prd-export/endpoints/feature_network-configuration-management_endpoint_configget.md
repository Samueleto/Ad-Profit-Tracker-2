# Feature Network Configuration Management Endpoint ConfigGet

GET /api/networks/config/:networkId

## Description

REQUEST: GET — networkId as URL path parameter. Bearer token in Authorization header. PROCESS: 1. Verify Firebase ID token — return 401 if invalid. 2. Validate networkId path param is one of the 4 supported networks — return 400 if not. 3. Fetch the networkConfigs document for this user+networkId from Firestore. 4. If document does not exist, return default config values for this network (isActive=true, syncSchedule='daily', timeoutSeconds=30, retryAttempts=3). 5. Return the config document. SUCCESS (200): { networkId, isActive, syncSchedule, dataRole, endpointOverride, timeoutSeconds, retryAttempts, lastSyncedAt, lastSyncStatus, lastSyncError, displayOrder, createdAt, updatedAt } ERRORS: 400 (invalid networkId), 401 (unauthenticated), 500 (Firestore read failure). SECURITY: UID from verified token only. Returns only the authenticated user's config.

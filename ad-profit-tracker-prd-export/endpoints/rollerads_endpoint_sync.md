# Rollerads Endpoint Sync

POST /api/networks/rollerads/sync

## Description

REQUEST: POST { dateFrom?: string (YYYY-MM-DD, defaults to yesterday), dateTo?: string (YYYY-MM-DD, defaults to yesterday) }
PROCESS:
1. Verify Firebase ID token via admin.auth().verifyIdToken(token) — return 401 immediately if invalid. uid from verified token only.
2. Validate dateFrom/dateTo if provided: must be valid YYYY-MM-DD strings, dateFrom <= dateTo, range max 90 days. Return 400 if invalid.
3. Check rate limit: max 10 manual syncs/hour per user. Return 429 if exceeded.
4. Fetch networkConfigs document for userId+networkId='rollerads'. If isActive=false, return 400 with message 'RollerAds network is disabled'.
5. Fetch encrypted API key from apiKeys collection where userId==uid AND networkId=='rollerads'. Return 404 if not found.
6. Decrypt API key using server-side AES/cryptr helper (same secret as /api/keys/save). Never log or return the decrypted key.
7. Retrieve endpointOverride and timeoutSeconds from networkConfigs (defaults: https://api.rollerads.com, 30s timeout).
8. Using axios (axios ^1.6.0), create a preconfigured instance: axios.create({ baseURL: endpointOverride || 'https://api.rollerads.com', headers: { Authorization: `Bearer ${decryptedKey}`, 'Content-Type': 'application/json' }, timeout: timeoutSeconds * 1000 }). Call GET /sites/stat with query params: date_from=dateFrom, date_to=dateTo, group_by=country. Record start time for latency measurement.
9. On API success: parse response.data rows. For each row (date + country combination), upsert a document into adStats collection: set userId, networkId='rollerads', date, country (ISO alpha-2 from response), revenue (from RollerAds earnings/revenue field), impressions, clicks, ctr, cpm (if provided), rawResponse (sanitized — strip any key/token/secret/authorization fields), syncedAt=serverTimestamp(). Use Firestore batch writes in chunks of 500 for efficiency. Cost field set to null (RollerAds is revenue-only).
10. Update networkConfigs document: lastSyncedAt=serverTimestamp(), lastSyncStatus='success', lastSyncError=null.
11. Write audit log directly to auditLogs Firestore collection via Admin SDK (not HTTP) with action='manual_sync_triggered', resourceType='sync', resourceId='rollerads', metadata={ triggered: ['rollerads'], latencyMs, rowsFetched: parsedRows.length, dateFrom, dateTo }, status='success'.
12. On RollerAds API error: update networkConfigs lastSyncStatus='failed', lastSyncError=errorMessage. Write audit log with status='failure', errorMessage. Return 502.
SUCCESS (200): { networkId: 'rollerads', rowsFetched: number, dateFrom: string, dateTo: string, latencyMs: number, syncedAt: timestamp }
ERRORS:
- 400: invalid date range, range exceeds 90 days, network disabled
- 401: invalid/expired Firebase ID token
- 404: no RollerAds API key saved
- 429: rate limit exceeded (10/hour per user)
- 502: RollerAds API unreachable or returned error
- 500: decryption failure, Firestore write failure
SECURITY: API key decrypted server-side only, never returned or logged. userId always from verified token. Metadata sanitized before audit log write. Rate limited to prevent runaway RollerAds API calls and Firestore write costs.

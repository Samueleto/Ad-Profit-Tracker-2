Apply header row styling: bold font, background fill using hex #4F46E5 (indigo), white font color. Freeze top row on each sheet (worksheet.views = [{state:'frozen',ySplit:1}]). Auto-fit column widths by setting worksheet.columns with width estimates. 10. Generate filename: use provided filename or default to 'ad-profit-tracker-export-{dateFrom}-{dateTo}'. 11. Write workbook to buffer using workbook.xlsx.writeBuffer(). SUCCESS (200): Binary .xlsx file stream with appropriate headers. ERRORS: 400 (invalid params), 401 (unauthenticated), 404 (no data for range), 429 (rate limit), 500 (generation failure). SECURITY: userId always from verified token. All Firestore queries scoped to authenticated user. Filename sanitized server-side. No API keys or encrypted values in export.

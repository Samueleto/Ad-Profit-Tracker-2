# Error Handling Endpoint UpdateRetryConfig

PATCH /api/errors/retry-config

## Description

REQUEST: PATCH { networkId: string (required), retryAttempts?: number (1–5), timeoutSeconds?: number (5–60) }. Bearer token in Authorization header. Updates the retry configuration for a specific network. This is a convenience wrapper that updates the retryAttempts and timeoutSeconds fields on the networkConfigs document — the same fields already managed by /api/networks/config/update, but surfaced here for the error handling context.
PROCESS:
1. Verify Firebase ID token via admin.auth().verifyIdToken(token) — return 401 if invalid. uid from verified token only.
2. Validate networkId: must be one of the 4 supported networks — return 400 if not.
3. Validate retryAttempts if provided: must be integer 1–5 — return 400 if out of range.
4. Validate timeoutSeconds if provided: must be integer 5–60 — return 400 if out of range.
5. At least one of retryAttempts or timeoutSeconds must be provided — return 400 if both omitted.
6. Check rate limit: max 20 config updates/hour per user — return 429 if exceeded.
7. Fetch networkConfigs document for userId+networkId — return 404 if not found.
8. Build update payload with only the provided fields plus updatedAt=serverTimestamp().
9. Update the networkConfigs document using Firestore Admin SDK.
10. Write audit log entry to auditLogs via Admin SDK: action='network_config_updated', resourceType='network_config', resourceId=networkId, metadata={ changedFields: { retryAttempts: { from, to }, timeoutSeconds: { from, to } } }, status='success'.
11. Return the updated config fields.
SUCCESS (200): { networkId: string, retryAttempts: number, timeoutSeconds: number, updatedAt: timestamp }
ERRORS:
- 400: invalid networkId, retryAttempts out of range, timeoutSeconds out of range, no fields provided
- 401: unauthenticated
- 404: networkConfigs document not found
- 429: rate limit exceeded
- 500: Firestore write failure
SECURITY: userId from verified token. Users can only update their own network configs. Audit logged.

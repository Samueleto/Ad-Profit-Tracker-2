# Cloudrun Endpoint Readiness Check

GET /api/ready

## Description

REQUEST: GET — no auth required. Used by Cloud Run readiness probe to determine if the instance is ready to serve traffic.
PROCESS:
1. No auth check — public readiness probe required by Cloud Run startup configuration.
2. Verify that required environment variables are set: FIREBASE_PROJECT_ID, INTERNAL_SYNC_SECRET, EMAIL_PROVIDER_API_KEY, EMAIL_FROM_ADDRESS. Return 503 if any are missing.
3. Verify Firebase Admin SDK is initialized (check admin.apps.length > 0). Return 503 if not initialized.
4. Return 200 if all checks pass.
SUCCESS (200): { ready: true, checks: { envVars: 'ok', firebaseAdmin: 'ok' } }
NOT READY (503): { ready: false, failedCheck: string }
ERRORS:
- 503: missing env vars or Firebase Admin not initialized
SECURITY: Public endpoint. Never reveals actual env var values — only reports presence/absence. Required by Cloud Run startup probe.

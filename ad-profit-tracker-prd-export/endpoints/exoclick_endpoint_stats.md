# Exoclick Endpoint Stats

GET /api/networks/exoclick/stats

## Description

REQUEST: GET with query params: dateFrom (YYYY-MM-DD, required), dateTo (YYYY-MM-DD, required), groupBy? ('day'|'total', default 'day').
PROCESS:
1. Verify Firebase ID token — return 401 if invalid.
2. Validate dateFrom and dateTo: required, valid YYYY-MM-DD, dateFrom <= dateTo, max 90-day range. Return 400 if invalid.
3. Query adStats Firestore collection: filter userId==uid AND networkId=='exoclick' AND date>=dateFrom AND date<=dateTo. Order by date ASC. Use getDocs (one-time read, not onSnapshot — data is not real-time).
4. If groupBy='total': aggregate all rows into a single totals object (sum cost, impressions, clicks; compute average ctr and cpm).
5. If groupBy='day': group rows by date, summing cost/impressions/clicks across all countries per day.
6. Return structured response.
SUCCESS (200): { networkId: 'exoclick', dateFrom, dateTo, groupBy, data: [{ date: string, cost: number, impressions: number, clicks: number, ctr: number, cpm: number }], totals: { cost: number, impressions: number, clicks: number } }
ERRORS:
- 400: missing/invalid date params, range exceeds 90 days
- 401: unauthenticated
- 500: Firestore read failure
SECURITY: userId always from verified token. Query scoped to authenticated user's documents only.

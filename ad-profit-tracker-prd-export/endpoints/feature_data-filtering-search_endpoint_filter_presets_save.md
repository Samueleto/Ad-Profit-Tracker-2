# Feature Data Filtering Search Endpoint Filter Presets Save

POST /api/filters/presets

## Description

REQUEST: POST { name: string (required, 1–50 chars), networks: string[] (required, subset of 4 networks), countries: string[] (required, array of ISO alpha-2 codes, max 50), metric: string (required, one of allowed metric values), dataQuality: string (required, one of 'all'|'anomalies'|'clean') }.
PROCESS:
1. Verify Firebase ID token via admin.auth().verifyIdToken(token) — return 401 if invalid. uid from verified token only.
2. Validate all fields: name 1–50 chars, networks subset of ['exoclick','rollerads','zeydoo','propush'], countries max 50 ISO codes, metric one of allowed values, dataQuality one of allowed values. Return 400 if invalid.
3. Check rate limit: max 20 preset saves/hour per user — return 429 if exceeded.
4. Read users Firestore document for uid. Check current savedFilterPresets array length — enforce max 20 presets per user (return 400 with 'Maximum 20 presets allowed' if at limit).
5. Generate a new preset ID (UUID). Build preset object: { id, name, networks, countries, metric, dataQuality, createdAt: serverTimestamp() }.
6. Use Firestore Admin SDK arrayUnion to append the new preset to users.savedFilterPresets array. Set users.updatedAt=serverTimestamp().
7. Write audit log entry to auditLogs Firestore collection via Admin SDK: action='preferences_updated', resourceType='preferences', resourceId='filter_preset', metadata={ presetId: id, presetName: name }, status='success'.
8. Return the created preset.
SUCCESS (201): { preset: { id: string, name: string, networks: string[], countries: string[], metric: string, dataQuality: string, createdAt: timestamp } }
ERRORS:
- 400: invalid fields, name too long, too many countries, max presets reached
- 401: invalid/expired Firebase ID token
- 429: rate limit exceeded (20/hour)
- 500: Firestore write failure
SECURITY: userId from verified token. Writes to authenticated user's users document only. Preset count capped at 20 to prevent document size abuse. Audit logged.

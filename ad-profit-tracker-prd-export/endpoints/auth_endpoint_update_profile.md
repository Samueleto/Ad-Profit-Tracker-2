# Auth Endpoint Update Profile

## Description

REQUEST: PATCH { displayName?: string, preferences?: { defaultDateRange?: string, timezone?: string } }
PROCESS: 1. Verify Firebase ID token. 2. Validate fields — displayName max 100 chars, preferences keys whitelisted. 3. Merge-update users/{uid} document with provided fields + updatedAt timestamp. 4. Return updated document.
SUCCESS (200): { uid, displayName, preferences, lastLoginAt }
ERRORS: 400 (invalid field values or unknown keys), 401 (invalid token), 404 (user not found), 500 (Firestore write failure)
SECURITY: Only updates own document. Whitelisted fields only — cannot update uid, email, role, or authProvider via this endpoint.

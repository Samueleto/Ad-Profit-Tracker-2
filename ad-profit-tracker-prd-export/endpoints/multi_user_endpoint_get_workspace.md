# Multi User Endpoint Get Workspace

GET /api/team/workspace

## Description

REQUEST: GET — no query params. Bearer token in Authorization header. Returns workspace metadata for the authenticated user's workspace.
PROCESS:
1. Verify Firebase ID token via admin.auth().verifyIdToken(token) — return 401 if invalid. uid from verified token only.
2. Read users Firestore document for uid. Extract users.workspaceId and users.workspaceRole.
3. If workspaceId is null, return 404.
4. Read the workspace owner's users document (workspaceId is the owner's uid) to get workspaceName, workspaceCreatedAt, and member count.
5. Return workspace metadata.
SUCCESS (200): { workspaceId: string, workspaceName: string, ownerUid: string, memberCount: number, createdAt: timestamp, currentUserRole: string }
ERRORS:
- 401: unauthenticated
- 404: no workspace found
- 500: Firestore read failure
SECURITY: userId from verified token. Workspace data scoped to authenticated user's workspaceId.

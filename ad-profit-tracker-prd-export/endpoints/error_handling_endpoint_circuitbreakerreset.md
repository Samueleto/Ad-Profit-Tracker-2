# Error Handling Endpoint CircuitBreakerReset

POST /api/errors/circuit-breaker/reset

## Description

REQUEST: POST { networkId: string (required — one of the 4 networks) }. Bearer token in Authorization header. Manually resets the circuit breaker for a specific network, allowing sync attempts to resume immediately.
PROCESS:
1. Verify Firebase ID token via admin.auth().verifyIdToken(token) — return 401 if invalid. uid from verified token only.
2. Validate networkId: must be one of ['exoclick','rollerads','zeydoo','propush'] — return 400 if not.
3. Check rate limit: max 10 circuit breaker resets/hour per user — return 429 if exceeded. Prevents users from bypassing the circuit breaker by repeatedly resetting it.
4. Fetch networkConfigs document for userId+networkId. If document doesn't exist, return 404.
5. If circuitBreakerOpen==false, return 400 with message 'Circuit breaker is not open for this network — no reset needed'.
6. Update networkConfigs document: circuitBreakerOpen=false, circuitBreakerOpenedAt=null, retryCount=0, nextRetryAt=null, lastSyncStatus='pending' (signals that a fresh sync attempt is warranted), lastSyncError=null, lastErrorCode=null. Use serverTimestamp() for updatedAt.
7. Write audit log entry directly to auditLogs Firestore collection via Admin SDK: action='network_config_updated', resourceType='network_config', resourceId=networkId, metadata={ action: 'circuit_breaker_reset', previousRetryCount: previousRetryCount, previousErrorCode: previousErrorCode, resetBy: 'user' }, status='success'.
8. Return confirmation with the reset network state.
SUCCESS (200): { networkId: string, circuitBreakerOpen: false, retryCount: 0, resetAt: timestamp, message: 'Circuit breaker reset. Sync attempts will resume on next trigger.' }
ERRORS:
- 400: invalid networkId, circuit breaker not open
- 401: unauthenticated
- 404: networkConfigs document not found for this network
- 429: rate limit exceeded (10/hour — prevents bypass abuse)
- 500: Firestore write failure
SECURITY: userId from verified token. Users can only reset their own network circuit breakers. Audit logged for accountability.

# Reconciliation Endpoint Resolve

PATCH /api/reconciliation/resolve

## Description

REQUEST: PATCH { documentIds: string[] (required, array of adStats document IDs, max 50), resolution: 'acknowledged'|'corrected'|'false_positive' (required), note?: string (optional, max 500 chars) }
PROCESS:
1. Verify Firebase ID token — return 401 if invalid. uid from verified token only.
2. Validate documentIds: required, non-empty array, max 50 items. Validate resolution enum. Validate note length if provided. Return 400 if invalid.
3. Check rate limit: max 20 resolve operations/hour per user. Return 429 if exceeded.
4. For each documentId, fetch the adStats document from Firestore. Verify userId==uid on each document — skip any that don't belong to the authenticated user (security: never return 403 for individual docs, just skip and report).
5. For each verified document, update: isReconciled=true, reconciledAt=serverTimestamp(), validationStatus='valid' (mark as resolved), anomalyFlags updated to add a resolution object to each flag: { resolvedAt: serverTimestamp(), resolution: resolution, note: note||null, resolvedBy: uid }.
6. Decrement networkConfigs.anomalyCount for each affected networkId by the count of resolved documents for that network. Use Firestore Admin SDK batch writes for all updates in chunks of 500.
7. Write one audit log entry to auditLogs via Admin SDK: action='network_config_updated', resourceType='reconciliation', resourceId='multiple'||networkId (if single network), metadata={ resolvedCount: number, resolution, documentIds, affectedNetworks: string[] }, status='success'.
8. Return resolution summary.
SUCCESS (200): { resolved: number, skipped: number, documentIds: string[], resolution: string }
ERRORS:
- 400: missing documentIds, invalid resolution, note too long, more than 50 IDs
- 401: unauthenticated
- 429: rate limit exceeded (20/hour)
- 500: Firestore batch write failure
SECURITY: userId from verified token. Each document's userId field verified before update. Batch scoped to authenticated user's documents only. Audit logged.

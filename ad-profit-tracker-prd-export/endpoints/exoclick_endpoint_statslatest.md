# Exoclick Endpoint StatsLatest

GET /api/networks/exoclick/stats/latest

## Description

REQUEST: GET — no query params. Returns the most recently synced day's aggregate stats for ExoClick, used for dashboard summary cards.
PROCESS:
1. Verify Firebase ID token — return 401 if invalid.
2. Query adStats Firestore collection: filter userId==uid AND networkId=='exoclick'. Order by date DESC. Limit to 1 document to find the latest date. Use getDocs.
3. If no documents found, return 404.
4. Using the latest date found, query all adStats documents for userId==uid AND networkId=='exoclick' AND date==latestDate to get all country rows for that day.
5. Aggregate: sum cost, impressions, clicks across all countries. Compute average ctr and cpm.
6. Also fetch networkConfigs document for userId+networkId='exoclick' to include lastSyncedAt and lastSyncStatus in the response.
SUCCESS (200): { networkId: 'exoclick', latestDate: string, cost: number, impressions: number, clicks: number, ctr: number, cpm: number, countryCount: number, lastSyncedAt: timestamp, lastSyncStatus: string }
ERRORS:
- 401: unauthenticated
- 404: no ExoClick data found — sync has not been run yet
- 500: Firestore read failure
SECURITY: userId from verified token. Query scoped to authenticated user only.
